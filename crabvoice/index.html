What is CrabVoice?

CrabVoice is a free, open-source application that provides real-time voice-over translation for online video content. It enables you to watch videos from 55+ platforms in your preferred language with synchronized dubbed audio.

Whether you're watching a YouTube tutorial, a Twitch stream, a VK video, or a TikTok clip — CrabVoice translates and dubs the audio in real-time, so you can understand content in any language without waiting for subtitles or manual translations.

Built with Rust and Tauri v2, CrabVoice is fast, lightweight, and works across Windows, Linux, and Android.

Key Features

  • 🎙️ Real-time Dubbing Automatic voice translation synchronized with video playback
  • 🌐 55+ Platforms YouTube, Twitch, VK, TikTok, Vimeo, Dailymotion and more
  • ⏭️ SponsorBlock Automatic skipping of sponsors and non-content segments
  • 🔒 Proxy Support HTTP and SOCKS5 proxy with connectivity verification
  • ⚡ Smart Playback Coordinated audio, pause, seeking, and speed controls

FAQ

Is CrabVoice free?

Yes, CrabVoice is completely free and open-source, released under the GPL-3.0 license.

What platforms does CrabVoice support?

CrabVoice supports 55+ video platforms including YouTube, Twitch, VK Video, TikTok, Vimeo, Dailymotion, and many more.

What operating systems can run CrabVoice?

CrabVoice is available for Windows, Linux (AppImage, deb, rpm), and Android.

What languages does CrabVoice translate to?

CrabVoice uses the Yandex Translation API and supports translation between numerous languages. The available language pairs depend on the translation service capabilities.

How does CrabVoice work?

CrabVoice intercepts video audio in real-time, sends it for translation via the Yandex API, and plays back the dubbed audio synchronized with the original video. It works as a desktop application built with Rust and Tauri v2.
